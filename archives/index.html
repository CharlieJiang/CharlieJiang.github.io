冒泡排序算法

算法基础——冒泡排序算法（Bubble Sort）原理、Java实现核心思想 对于一组需要排序的数字，依次将每个位置上的数字与后面位置上的数字进行比较，如果两个数字的顺序错误，则调换两个位置上的数字。 算法原理 以降序排序为例 假设有n个数字的数组a，需要进行n-1轮的排序，每轮确定一个数字的具体位置 第一轮：将a[0]~a[n-1]之间的数字两两相邻的依次进行比较，当后面的数字大于前面的数字时，两个数字互换，此轮结束，a[n-1]则为最小的数字。 第二轮：将a[0]~a[n-2]之间的数字两两相邻的依次进行比较，当后面的数字大于前面的数字时，两个数字互换，此轮结束，a[n-2]则为第2小的数字。 … 第n-1轮：将a[0]~a[1]之间的数字两两相邻的依次进行比较，当后面的数字大于前面的数字时，两个数字互换，此轮结束，a[1]则为第n-1小的数字。 1234567891011121314151617181920212223242526272829/** * 冒泡排序：以降序排序为例。 * 先找最小数，相邻两个数字依次进行比较 * @param a * @return */public int[] bubbleSort_02(int[] a){ if(null == a || a.length <= 0){ return null; } // 计算数字个数 int n = a.length; // 临时变量：两个数字交换值时使用 int temp; // 共n-1轮排序的for循环 for(int i = 0; i < n; i++){ //第i轮排序的for循环 for(int j = 1; j < n-i; j++){ //前值小于后值时，两者互换 if(a[j-1] < a[j]){ temp = a[j-1]; a[j-1] = a[j]; a[j] = temp; } } } return a;} 算法的时间复杂度 O(N²) 算法特点 冒泡排序是一种稳定的排序算法。排序过程中，遇到相同的元素并不会交换，从而保证了相同元素的前后位置关系不变。
