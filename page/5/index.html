Java引用与方法引用详解

发表于 2020-11-30 分类于 Java ， Java基础

引用介绍

引用也称对象引用（Object Reference），Java中的引用类似于C++ 中的指针，会指向对象的存储空间地址，需要使用引用来访问对象。

引用级别

在JDK1.2中，加入了四种不同级别的引用，等级由高到低依次是：强引用、软引用、弱引用、虚引用。
引用级别的高低决定了Java垃圾回收器在内存不足时对对象进行回收操作的优先顺序，级别越高，越不容易被回收。

final修饰符

final修饰符的意思是最终的、不可变的，一般与static修饰符一起用于创建静态常量。

面向对象三大特性

封装、继承、多态

封装

隐藏对象的属性和具体的实现细节，对外提供属性的访问方法，以控制对象属性读写的访问级别。
例如我们经常使用的实体类就是比较典型的封装：

  • 实体类的属性一般设置为private，不允许外部类直接调用，即对外部类隐藏具体属性。
  • 实体类的getter和setter方法为外部类提供了访问实体类属性的方法，即外部类可通过实体类方法间接访问其属性。
