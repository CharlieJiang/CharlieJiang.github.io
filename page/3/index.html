开发中的幂等性问题

发表于 2021-04-25 分类于 其它

幂等概念：对于同一个系统，在同样条件下，一次请求和多次重复请求对系统资源的影响是一致的，就称该操作是幂等的。

开发中的幂等性：

幂等性是系统接口对外的一种承诺，而不是实现，承诺多次操作的结果都会是一样的。
例如：根据id删除数据库记录这一操作，因为id是唯一的，所以这一操作执行一次或是多次，对数据库而言结果都是一样的，都只会删除相同的一条记录。

SPDY协议

发表于 2021-04-23 分类于 网络协议

SPDY是谷歌开发的下一代网络协议，它并不是一种用于替换HTTP的协议，而是对HTTP协议的增强。以下是SPDY提供的对HTTP的四项改进：
  1、多路复用
  单个SPDY连接内可以并发地发送多个请求，并且不限制请求数量。
  2、请求优先级
  SPDY允许给每个请求设置优先级。这样高优先级的请求就可以优先响应，也避免了多个请求并发时重要的请求得不到优先响应。
  3、header压缩
  4、基于https的加密协议传输，大大提高了传输数据的可靠性。

TCP报文中的Payload

发表于 2021-04-23 分类于 网络协议

  Payload是什么？简单来说，就是数据帧或数据包中的原始数据。
  通常在传输数据时，为了使数据传输更可靠，要把原始数据分批传输，并且在每一批数据的头和尾都加上一定的辅助信息，比如数据量的大小、校验位等。这样就相当于给已经分批的原始数据加一些外套，这些外套起标识作用，使得原始数据不易丢失。一批数据加上“外套”就形成了传输通道的基本传输单元，叫做数据帧或数据包，而其中的原始数据就是payload，称作有效载荷。
