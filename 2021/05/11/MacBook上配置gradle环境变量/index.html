MacBook上配置gradle环境变量

发表于 2021-05-11 分类于 其它 阅读次数：

查找gradle路径

  • 自动下载：Android Studio自已自动下载的gradle，路径一般位于/Users/用户名/.gradle/wrapper/dists下，配置时取的路径为/Users/用户名/.gradle/wrapper/dists/gradle-5.6.4-all/ankdp27end7byghfw1q2sw75f/gradle-5.6.4。
  • 手动下载：需要根据下载文件的位置确定gradle路径

配置环境变量

MacBook上配置各种环境变量，就是将路径信息按指定规则保存到一个.bash_profile文件中。如Java环境变量、Android环境变量、Python环境变量等。

配置gradle环境变量步骤如下：

  1. 打开命令行窗口
  2. 指定文本编辑器打开.bash_profile文件：命令行输入open -e .bash_profile
  3. 如果文件不存在，则创建：命令行输入 touch .bash_profile
  4. 编辑.bash_profile文件，输入以下内容： 
    GRADLE_HOME=/Users/用户名/.gradle/wrapper/dists/gradle-5.6.4-all/ankdp27end7byghfw1q2sw75f/gradle-5.6.4
export GRADLE_HOME
export PATH=${PATH}:${GRADLE_HOME}/bin
  5. 保存.bash_profile文件：命令行输入 source ~/.bash_profile
  6. 校验gradle环境变量配置：命令行输入 gradle -version
