git修改访问Github的方式为token

发表于 2021-08-22 分类于 git

设置方式

1、在Github创建Personal Access Token
按步骤访问：Github =>Settings =>Developer Settings => Personal Access Token => Generate New Token
或者直接访问https://github.com/settings/tokens/new
创建token成功后复制token

2、Windows上修改密码访问Github为Token访问
打开Windows设置 => 搜索“Windows 凭据” =>管理 Windows 凭据 =>找到”git:https://github.com“ => 编辑 =>密码中输入刚创建的token =>保存

圆角进度条实现：基于ProgressBar+自定义进度条样式

发表于 2021-08-15 分类于 Android ， 自定义View

1、创建drawable文件配置进度条背景和圆角半径：progressbar_bg 

<?xml version="1.0" encoding="utf-8"?><!--自定义横向圆角进度条-->
<layer-list xmlns:android="http://schemas.android.com/apk/res/android">
    <!--进度条背景色-->
    <item android:id="@android:id/background">
        <shape>
            <corners android:radius="5dip" />
            <solid android:color="#FEF2E1" />
        </shape>
    </item>
    <!--进度条颜色-->
    <item android:id="@android:id/progress">
        <scale android:scaleWidth="100%">
            <shape>
                <corners android:radius="5dip" />
                <solid android:color="#FEAF3C" />
            </shape>
        </scale>
    </item>
</layer-list>

进度条圆角半径设置注意：radius需要根据实际进度条的宽度来取值，一般需要大于进度条宽度的一半。

Android中aar文件和jar文件的区别

发表于 2021-08-08 分类于 Android

jar文件：jar文件就是 Java Archive File，即java归档文件，是与平台无关的文件格式，将许多文件组合成一个压缩文件。Android项目打包的jar文件只包含class文件和清单文件，无资源文件。
aar文件：aar文件就是Android Archive，是一个Android库项目的二进制归档文件。Android库项目打包的aar文件包含所有的class文件和资源文件。

GIT与SVN区别

发表于 2021-08-08 分类于 git ， svn

主要区别

  1、最核心的区别Git是分布式的，而Svn不是分布的。Git更倾向于分布式开发，因为每一个开发人员的电脑上都有一个Local Repository,所以即使没有网络也一样可以Commit，查看历史版本记录，创建项 目分支等操作，等网络再次连接上Push到Server端。
  2、Git把内容按元数据方式存储，而SVN是按文件：因为,.git目录是处于你的机器上的一个克隆版的版本库，它拥有中心版本库上所有的东西，例如标签，分支，版本记录等。
  3、 Git没有一个全局版本号，而SVN有：目前为止这是跟SVN相比Git缺少的最大的一个特征。
