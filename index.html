final修饰符

final修饰符的意思是最终的、不可变的，一般与static修饰符一起用于创建静态常量。

  • 作用范围：
      修饰变量（类变量、局部变量、形参）、方法和类

  • 修饰类：
      final修饰的类不能被继承，即无法拥有自己的子类。
      如果对一个final修饰的类进行继承，在编译期间会发生错误。

发表于 2020-11-23 更新于 2020-11-26 分类于 Java ， Java基础

面向对象三大特性

封装、继承、多态

封装

隐藏对象的属性和具体的实现细节，对外提供属性的访问方法，以控制对象属性读写的访问级别。
例如我们经常使用的实体类就是比较典型的封装：

  • 实体类的属性一般设置为private，不允许外部类直接调用，即对外部类隐藏具体属性。
  • 实体类的getter和setter方法为外部类提供了访问实体类属性的方法，即外部类可通过实体类方法间接访问其属性。
